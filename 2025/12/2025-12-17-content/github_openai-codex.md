## Quickstart

### Installing and running Codex CLI

Install globally with your preferred package manager. If you use npm:

```
npm install -g @openai/codex
```

Alternatively, if you use Homebrew:

```
brew install --cask codex
```

Then simply run `codex` to get started:

```
codex
```

If you're running into upgrade issues with Homebrew, see the [FAQ entry on brew upgrade codex](/openai/codex/blob/main/docs/faq.md#brew-upgrade-codex-isnt-upgrading-me).

You can also go to the [latest GitHub Release](https://github.com/openai/codex/releases/latest) and download the appropriate binary for your platform.

Each GitHub Release contains many executables, but in practice, you likely want one of these:

* macOS
  + Apple Silicon/arm64: `codex-aarch64-apple-darwin.tar.gz`
  + x86\_64 (older Mac hardware): `codex-x86_64-apple-darwin.tar.gz`
* Linux
  + x86\_64: `codex-x86_64-unknown-linux-musl.tar.gz`
  + arm64: `codex-aarch64-unknown-linux-musl.tar.gz`

Each archive contains a single entry with the platform baked into the name (e.g., `codex-x86_64-unknown-linux-musl`), so you likely want to rename it to `codex` after extracting it.

### Using Codex with your ChatGPT plan

[![Codex CLI login](/openai/codex/raw/main/.github/codex-cli-login.png)](/openai/codex/blob/main/.github/codex-cli-login.png)

Run `codex` and select **Sign in with ChatGPT**. We recommend signing into your ChatGPT account to use Codex as part of your Plus, Pro, Team, Edu, or Enterprise plan. [Learn more about what's included in your ChatGPT plan](https://help.openai.com/en/articles/11369540-codex-in-chatgpt).

You can also use Codex with an API key, but this requires [additional setup](/openai/codex/blob/main/docs/authentication.md#usage-based-billing-alternative-use-an-openai-api-key). If you previously used an API key for usage-based billing, see the [migration steps](/openai/codex/blob/main/docs/authentication.md#migrating-from-usage-based-billing-api-key). If you're having trouble with login, please comment on [this issue](https://github.com/openai/codex/issues/1243).

### Model Context Protocol (MCP)

Codex can access MCP servers. To configure them, refer to the [config docs](/openai/codex/blob/main/docs/config.md#mcp_servers).

### Configuration

Codex CLI supports a rich set of configuration options, with preferences stored in `~/.codex/config.toml`. For full configuration options, see [Configuration](/openai/codex/blob/main/docs/config.md).

### Execpolicy

See the [Execpolicy quickstart](/openai/codex/blob/main/docs/execpolicy.md) to set up rules that govern what commands Codex can execute.
